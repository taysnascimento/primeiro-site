Site Séries/ Filmes

Este É um site de Filmes e Séries onde os usuários podem navegar,pesquisar e visualizar informações sobre os filmes e séries, como sinopse, trailers, elenco e ultimos lansamentos. E adicionalos a uma lista de favoritos.

Descrição do Site

Este projeto e um site desenvolvido para permitir que os usuários possam acessar multiplas informaçoes sobre diversos filmes e séries dentro de um mesmo site. Ele pode ser acessado pela seguinte URL digitada no navegador (http://127.0.0.1:5500/Index.html)

Funcionalidades

  • Listagem de filmes e séries atualizadas
  • Exibição de detalhes de cada filme e série
  • pesquisa por titulos e categori.
  • exibição de traileres e sinopse de cada filme e série
  • menu de listagem com filmese séries favoritos

Tecnologias Utilizadas

  • HTML5
  • Css
  • Github

Como Acessar o Site

  1. Digite a seguinte URL em seu navegador (http://127.0.0.1:5500/Index.html).
  2. Ao abrir a página inicial do site escolha um dos manus acima.
  3. Escolha a categoria ou digite o titulo que deseja.
  4. Ao digitar o que deseja, o usúario sera direcionado a uma nova tela do site.
  5. esta nova tela do site vai conter, filme/ série que foi escolhido, sinopse, classificação e titulos realacionados.
  6. Agora e só desfutar da sua escolha, e continuar explorando o que o site tem a lhe oferecer.
