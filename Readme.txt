# Site Series/ Filmes

Este É um site de Filmes e Séries onde os usuários podem navegar,pesquisar e visualizar informações sobre os filmes e séries, como sinopse, trailers, elenco e ultimos lansamentos. E adicionalos a uma  lista de favoritos.

## Descrição do Site

Este projeto e um site desenvolvido para permitir que os usuários possam acessar multiplas informaçoes sobre diversos filmes e séries. Ele pode ser acessado pela seguinte URL (http://127.0.0.1:5500/Index.html) 

## Funcionalidades

- Listagem de filmes e séries atualizada.
- Exibição de detalhes de cada filme.
- Pesquisa de filmes e séries por título.
- pesquisa por categoria.
- Exibição de trailers  e sinopse dos filmes e séries.

## Tecnologias Utilizadas

- HTML5
- CSS

## Como acessar o site

1.digite a seguinte URL (http://127.0.0.1:5500/Index.html)

2. Ao abrir a pagina escolha um dos menus acima.

3.Digite a categoria que deseja ou o titulo do filme ou serie.

4. Ao digitar o que deseja, será direcionado a uma nova tela do site.

5.esta nova tela do site vai conter, filme,sinopse,classificação e titulos realacionados.

6. Apos selecionar o que deseja,o que resta e apenas desfrutar.
